github2author extracts authors information from commit logs.

## Installation
```sh
➜  ~ go install -v github.com/kawakatz/github2author@latest
```

## Usage
```sh
➜  ~ export GITHUB_TOKEN=<PERSONAL ACCESS TOKEN>
➜  ~ github2author <UserName or OrgName>
```
